**# AI Career Recommendation System

## 📌 Overview
This project is an AI-based Career Recommendation System that suggests suitable career paths based on a user's skills and interests.

The system combines:
- Rule-based AI (decision-making logic)
- Machine Learning (KNN classification)

---

## 🎯 Problem Statement
Many students face confusion while choosing a career path due to lack of guidance. This system helps by analyzing user inputs and recommending suitable career options.

---

## 🧠 Concepts Used
- Artificial Intelligence (Rule-Based System)
- Machine Learning (Classification using KNN)
- Feature Engineering
- Decision Making
- Confidence Calculation

---

## ⚙️ How It Works
1. User enters skill ratings (0–10)
2. Rule-based system calculates scores
3. ML model predicts career
4. Final recommendation is generated

---

## ▶️ How to Run the Project

### Step 1: Install dependencies**
pip install scikit-learn

### Step 2: Run the program
python final_project.py
